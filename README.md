# Prime_Video_Analysis
This dashboard provides a simple analysis of Amazon Prime movies and TV shows data using Power BI. It helps to understand content trends, genres, ratings, and release years through interactive charts and visuals. The dashboard is designed to present important insights in an easy and clear way.
📌 Project Overview

This project is based on analyzing Amazon Prime movies and TV shows data using Power BI.
The goal is to understand content trends, ratings, genres, and release patterns in a simple and visual way.

🎯 Objective
To analyze Amazon Prime content data
To find trends in movies and TV shows
To create an interactive dashboard for easy understanding
🛠️ Tools & Technologies Used
Power BI (for dashboard creation)
Excel / CSV (for dataset)
Data Cleaning & Data Visualization
📂Dataset Information

The dataset includes:

Title of shows and movies
Genre
Release year
Ratings
Duration
Country
📈 Key Features of Dashboard
Total number of Movies & TV Shows
Content distribution by Genre
Year-wise release trend
Ratings analysis
Country-wise content availability
Interactive filters for better analysis
🔍 Insights Gained
Most content is released after 2010
Drama and Comedy are the most popular genres
Movies are more than TV shows on the platform
Some countries produce more content than others
💡 Why This Project?

This project helps to:

Improve data analysis skills
Learn Power BI visualization
Understand real-world streaming data trends
🚀 Conclusion

This dashboard makes it easy to explore Amazon Prime content in a simple and visual way. It helps users quickly understand patterns, trends, and important insights from the data.
